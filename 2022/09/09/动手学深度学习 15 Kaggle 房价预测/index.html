2.3 清洗数据

接下来对数据集整体进行处理。首先是对数据集进行数据清洗。将训练集和测试集合并得到完整数据集：

all_features = pd.concat((train_data.iloc[:, 1:-1], test_data.iloc[:, 1:]))

这里使用了 pandas 中的 concat() 函数，将训练集除去 Id 的部分和测试集进行了合并，得到了完整数据集。

之后对数值特征进行标准化，并填充所有空值项。正则化使用 \(z\) 值法，即假设样本 \(X\) 均值、方差均存在，且有 \(E(X)=\mu,\ D(X)=\sigma^2\) ，对样本按如下方式标准化： \[ z=\frac{X-\mu}{\sigma} \] 进行以上过程的代码如下：

# 获取所有数值特征的列索引
numeric_features = all_features.dtypes[all_features.dtypes != 'object'].index

# z值法标准化
all_features[numeric_features] = all_features[numeric_features].apply(
    lambda x : (x - x.mean()) / (x.std()))

# 填充空值
all_features[numeric_features] = all_features[numeric_features].fillna(0)

对于字符型数据，这里采用 one-hot 编码。对于 one-hot 编码的解释可以参考 数据预处理：独热编码（One-Hot Encoding）和 LabelEncoder标签编码 - 理想几岁 - 博客园 (cnblogs.com) 。pandas 中提供了 get_dummies() 函数用于实现独热编码。

all_features = pd.get_dummies(all_features, dummy_na=True)

3 建立网络

3.1 转换为张量

首先先形成训练特征、测试特征及训练标签的张量。

n_train = train_data.shape[0]

3.2 定义平方损失及线性网络

在 d2l 官网中对于定义线性模型的解释是这样的：

首先，我们训练一个带有损失平方的线性模型。 显然线性模型很难让我们在竞赛中获胜，但线性模型提供了一种健全性检查， 以查看数据中是否存在有意义的信息。 如果我们在这里不能做得比随机猜测更好，那么我们很可能存在数据处理错误。 如果一切顺利，线性模型将作为基线（baseline）模型， 让我们直观地知道最好的模型有超出简单的模型多少。

但确实是用 MLP 打不过这里给出的线性网络，可能是笔者对 MLP 的打开方式不对……

定义平方损失及输入特征维度：

loss = nn.MSELoss()
in_features = train_features.shape[1]

这里使用 MSE （均方误差，Mean Squared Error）作为损失函数，即 \[ \mathcal{L}=\frac 1 n \sum_{i=1}^n(y_i-\hat{y_i})^2 \]

通过 torch.nn 定义线性网络：

def get_net():
    net = nn.Sequential(nn.Linear(in_features,1))
    return net

这里的网络就只有一层全连接层，输入维度是数据集维度，输出维度是 \(1\) 。相当于以下的线性函数（\(n\) 为输入维度）： \[ y=w_1x_1+k_2w_2+\cdots+w_nx_n + b \]

3.3 对数均方根误差

对于房价、股票价格这种数据而言，我们更会关心相对变化，而绝不是绝对变化，因为这种数据的变化和差异可能会很悬殊，绝对差异没有任何意义。也就是说，我们更关心相对误差 \(\displaystyle \frac{y-\hat y}{y}\) ，而不是绝对误差 \(y-\hat y\) 。

解决以上问题的一种方法是使用对数来衡量差异。事实上，这也是这个比赛中用于评定结果的误差指标。

原本的误差 \(err\) 定义如下 \[ err:=\delta\quad \text{where } |y-\hat y|\le \delta \]

通过对准确值和预测值取对数，得到 \[ |\log y-\log \hat y|\le \delta \] 展开绝对值，得 \[ e^{-\delta}\le\frac{\hat y}{y}\le e^\delta \] 这样就得到了相对误差中的比值形式。那么，通过以上取对数的方式，就使得预测价格和真实价格的对数之间出现了如下的均方根误差： \[ \mathcal{L}=\sqrt{\frac 1 n\sum_{i=1}^n(\log y_i-\log \hat {y_i})^2} \] 定义对数均方根误差的代码如下：

def log_rmse(net, features, labels):
    clipped_preds = torch.clamp(net(features), 1, float('inf'))
    rmse = torch.sqrt(loss(torch.log(clipped_preds), torch.log(labels)))
    return rmse.item()

首先对数据进行裁剪，即将所有数据控制到 \([1,+\infty]\) 区间以内，使用 torch.clamp() 函数进行。之后根据公式定义对数均方根误差。

4 训练

训练的大致代码过程和前面几节中的实现类似，不同的是，这里采用了 Adam 优化器。Adam 的一大优点是对于初始学习率不是那么敏感。

训练函数的代码如下：

def train(net, train_features, train_labels, test_features, test_labels,
             num_epochs, learning_rate, weight_decay, batch_size):
    train_ls, test_ls = [], []
    train_iter = d2l.load_array((train_features, train_labels), batch_size)
    optimizer = torch.optim.Adam(net.parameters(), lr=learning_rate, 
                                 weight_decay=weight_decay)
    for epoch in range(num_epochs):
        for X, y in train_iter:
            optimizer.zero_grad()
            l = loss(net(X), y)
            l.backward()
            optimizer.step()
        train_ls.append(log_rmse(net, train_features, train_labels))
        if test_labels is not None:
            test_ls.append(log_rmse(net, test_features, test_labels))
    return train_ls, test_ls

通过训练函数，返回训练集损失和测试集损失（采用对数均方根损失）。

5 \(K\) 折交叉验证

前面的几节中涉及了有关 \(K\) 折交叉验证的知识，它能够有助于模型选择和超参数的调整。

首先定义函数，在 \(K\) 折交叉验证过程中返回第 \(i\) 折的数据，并以其作为验证数据，其余部分作为训练数据。

def get_k_fold_data(k, i, X, y):
    assert k > 1
    fold_size = X.shape[0] // k
    X_train, y_train = None, None
    for j in range(k):
        idx = slice(j * fold_size, (j + 1) * fold_size)
        X_part, y_part = X[idx, :], y[idx]
        if j == i:
            X_valid, y_valid = X_part, y_part
        elif X_train is None:
            X_train, y_train = X_part, y_part
        else:
            X_train = torch.cat([X_train, X_part], 0)
            y_train = torch.cat([y_train, y_part], 0)
    return X_train, y_train, X_valid, y_valid

当我们在 \(K\) 折交叉验证中训练 \(K\) 次过后，返回训练和验证误差的平均值。

def k_fold(k, X_train, y_train, num_epochs, learning_rate, weight_decay, batch_size):
    train_l_sum, valid_l_sum = 0, 0
    for i in range(k):
        data = get_k_fold_data(k, i, X_train, y_train)
        net = get_net()
        train_ls, valid_ls = train(net, *data, num_epochs, 
                                   learning_rate, weight_decay, batch_size)
        train_l_sum += train_ls[-1]
        valid_l_sum += valid_ls[-1]
        if i == 0:
            d2l.plot(list(range(1, num_epochs + 1)), [train_ls, valid_ls], 
                     xlabel='epoch', ylabel='rmse', xlim=[1, num_epochs + 1],
                     legend=['train', 'valid'], yscale='log')
        print(f'fold {i + 1}, train log rmse {float(train_ls[-1]):f}, '
              f'valid log rmse {float(valid_ls[-1]):f}')
    return train_l_sum / k, valid_l_sum / k

7 提交结果

根据前文定义的所有函数和选择的超参数，这里使用所有数据进行训练，并将这样方式获得的模型应用于测试集进行预测，并输出最终结果，到 submission.csv 文件中。

def train_and_pred(train_features, test_feature, train_labels, test_data,
                   num_epochs, lr, weigth_decay, batch_size):
    net = get_net()
    train_ls, _ = train(net, train_features, train_labels, None, None,
                        num_epochs, lr, weight_decay, batch_size)
    d2l.plot(np.arange(1, num_epochs + 1), [train_ls], xlabel='epoch', ylabel='log rmse',
