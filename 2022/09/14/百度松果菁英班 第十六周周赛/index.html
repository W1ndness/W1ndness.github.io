1 \(2\) 的 \(n\) 次幂

第一次纯签到题，直接暴力可 AC 。

Code

LANG=C++

#include <bits/stdc++.h>
using namespace std;

typedef long long LL;

int n;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);
    cin >> n;
    cout << (LL) pow(2, n);
    return 0;
}

2 个数统计

本题涉及阶乘，但难度较小，直接使用 Python 库函数 math.factorial() ，之后对得到的结果转字符串计数即可。

Code

LANG=Python3

import math

def main():
    n, a = map(int, input().split())
    s = str(math.factorial(n))
    print(s.count(str(a)))


if __name__ == '__main__':
    main();

3 数组扦插

本题思路比较简单。由于要求前半段升序排列为 \(a_i\) ，后半段逆序排列为 \(b_i\) ，划定数组中点序号容易出错，那么就对后半段逆向思维，将后半段升序排列，记为 \(b_i'\) 。此时数组变为一个两段升序数组，但当数组长度为奇数时涉及对中间值不变的特判。这里容易陷入的误区是，对于数组的处理必须是分别排序，当对后半段逆向思维改为升序排序后，容易误认为整个数组全部为升序。

于是对于数组的处理思路便明确下来，首先确定数组长度的奇偶性。若为奇数，则对除中间一位外对数组两半进行处理，若为偶数，则直接对数组两半进行处理。

接下来，由于我们已经将数组按照题目要求排序，则下一步进行“扦插”操作。这里就是一个首位双指针更新得到答案的过程了，不过也需要判断当长度为奇数时，不要漏掉中间位置不变的那一位。

Code

LANG=C++

#include <bits/stdc++.h>
using namespace std;

const int N = 100010;

int n;
int q[N];
int ans[N];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);
    cin >> n;
    for (int i = 0; i < n; i++) cin >> q[i];
    if (n % 2 == 0) {
        sort(q, q + n / 2);
        sort(q + n / 2, q + n);
    }
    else {
        sort(q, q + n / 2);
        sort(q + n / 2 + 1, q + n);
    }
    int i = 0, j = n - 1;
    int k = 0;
    while (i < j) {
        ans[k++] = q[i];
        ans[k++] = q[j];
        i++, j--;
    }
    if (i == j) ans[k++] = q[i];
    for (int i = 0; i < n; i++) cout << ans[i] << ' ';
    return 0;
}

4 MXR 竞赛

本题应该是这一次周赛最难的一道题了，因为要考虑的特殊情况实在是太多。

第一遍看到题的时候，第一思路是 DP 做的最大乘积子序列，但回忆之后发现，DP 解出的解应当是数组中连续的一段子数组。所以 DP 法无效。

之后转到思考在题目条件下，如何能够获得最多的积分。从数学的角度，对于一个序列而言，选取其中的一个子集，既然要求乘积最大，那么所有的正数是必须要选的。之后，我们考虑负数。由于负负得正，我们希望选取更多的偶数个负数，使之能够累积到乘积之上，使乘积继续变大。

经过以上分析后，本题的大体思路已经很明确了。首先统计所有正数、负数，答案中必含有所有正数的乘积，答案中也必定含有按绝对值排序的最多偶数个负数的乘积。

可能有些人觉得分析到这里之后已经结束了，可是你会发现按上面的思路并不能够 AC 。于是接下来考虑这道题有什么特殊情况。

首先是只有一个元素的时候，体重要求他只有解决一个问题，才能够出现在积分榜上，所以党只有一个元素时，答案是其本身。

其次，题目中并没有否认 \(0\) 元素的存在。在一般情况下，只要剔除 \(0\) 的存在，不会对答案产生什么影响。但两种极其特殊的情况如下：

0 0 0 0 0 0 ... 0 # 全0
0 0 0 -3 0 0 ... 0 # 只有一个负元素，其余全为0

由于我们涉及到连乘，所以初始值应当被设为 \(1\) 。对于以上两种情况而言，按照一般情况的处置方法，第一种情况会得到 \(1\) ，但其实正确答案应当为 \(0\) ；对于第二种情况也是如此。

所以，我们必须对以上两种情况进行特判。处理方式是在输入时判断是否有 \(0\) 元素的出现，在输入完成后，根据正数、负数元素的个数以及是否存在 \(0\) 元素进行综合判定是否属于以上两种特殊情况。

Code

LANG=C++

#include <bits/stdc++.h>

using namespace std;

typedef long long LL;

const int N = 100010;
const int MOD = 998244353;

int n;
int a[N];
vector<int> neg, pos;
bool hasZero;
LL ans = 1;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0); cout.tie(0);
    cin >> n;
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        if (a[i] < 0) neg.push_back(-a[i]);
        else if (a[i] > 0) pos.push_back(a[i]);
        else hasZero = true;
    }
    if (n == 1) {
        cout << a[0];
        return 0;
    }
    if (neg.size() <= 1 && pos.size() == 0 && hasZero) {
        ans = 0;
        cout << ans;
        return 0;
    }
    for (int i = 0; i < pos.size(); i++) ans = (ans * pos[i]) % MOD;
    sort(neg.begin(), neg.end());
    if (neg.size() % 2 == 0) {
        for (int i = 0; i < neg.size(); i++) ans = (ans * neg[i]) % MOD;
    }
    else {
        for (int i = 1; i < neg.size(); i++) ans = (ans * neg[i]) % MOD;
    }
    cout << ans % MOD;
    return 0;
}

5 小码哥的跳棋游戏

这道题算是一道思维题。记当前棋子的位置在 \(x\) ，其中 \(0\le x\le n-1,\ x\in\mathbb Z\) 。若位置 \(x\) 有棋子，则记判定函数 \(d(x)=1\) 。

当棋子在位置 \(x\) 时，有以下两种情况：

  • 若 \(d(x+1)=1\) 但 \(d(x+2)\neq1\) 时，直接跳过位置 \(x+1\) ，到达 \(x+2\) 。
  • 若 \(d(x+1)=1\) 但 \(d(x+2)=1\) 时，为满足花费能量最少，那么需要“能跳则跳”，即花费 \(1\) 点能量摧毁 \(x+2\) 的棋子，之后跳过 \(x+1\) ，到达 \(x+2\) 。

这里有一种特殊情况需要被考虑，当棋子到达位置 \(n-2\) 时，如果 \(d(n-1)=1\) ，那么如果要到达 \(n-1\) 就必须要被迫花费 \(1\) 点能量，摧毁 \(n-1\) 的棋子。

在实际处理的过程中，可以简化判断条件，即只判断当前位置和它的下一个位置的棋子情况。如果当前位置\(k\) 的 \(d(k)=1\) ，而又有 \(d(k+1)=1\) ，那么就必须摧毁 \(k+1\) 的棋子。这样做的好处是不用考虑游标会一次步进 \(2\) 的情况，而只需要一个 for 即可模拟小码哥棋子的位置。

Code

LANG=C++

#include <bits/stdc++.h>
using namespace std;

const int N = 100010;

int n;
int a[N];
int ans;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0); cout.tie(0);
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n - 1; i++) {
        if (a[i] == 1) {
            if (a[i + 1] == 1) {
                ans++;
                a[i + 1] = 0;
            }
        }
    }
    if (a[n - 1] == 1) ans++;
    cout << ans;
    return 0;
}

6 矩阵乘法

常规 \(O(n^3)\) 模拟即可 AC 。

Code

#include <bits/stdc++.h>
using namespace std;

const int N = 1010;

int l, m, n;
int a[N][N], b[N][N];
int ans[N][N];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);
    cin >> l >> m >> n;
    for (int i = 0; i < l; i++) {
        for (int j = 0; j < m; j++) {
            cin >> a[i][j];
        }
    }
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            cin >> b[i][j];
        }
    }
    for (int i = 0; i < l; i++) {
        for (int j = 0; j < n; j++) {
            for (int k = 0; k < m; k++) ans[i][j] += a[i][k] * b[k][j];
        }
    }
    for (int i = 0; i < l; i++) {
        for (int j = 0; j < n; j++) {
            cout << ans[i][j] << ' ';
        }
        cout << endl;
    }
    return 0;
}
