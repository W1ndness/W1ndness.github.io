一、组合数相关概念

求组合数 CabC_a^bC​a​b​​​​ 的公式是

\large\begin{align} C_a^b &=\frac{a(a-1)\cdots(a-b+1)}{1\times2\times3\times\cdots\times b}\\ &=\frac{a!}{b!(a-b)!} \end{align}

高中学习过组合数有如下性质：

Cab=Ca−1b+Ca−1b−1C_a^b=C_{a-1}^b+C_{a-1}^{b-1} C​a​b​​=C​a−1​b​​+C​a−1​b−1​​

相关推导过程可以查阅有关资料，于是通过上面的公式就可以通过递推求出每一个组合数的值。

三、卡特兰数

卡特兰数（Catalan number），又称明安图数。

卡特兰数 CnC_nC​n​​ 满足如下递推关系：

\begin{gather} C_{n+1}=C_0C_n+C_1C_{n-1}+\cdots+C_nC_0\\ (n-3)C_n=\frac n 2(C_3C_{n-1}+C_4C_{n-2}+C_5C_{n-3}+\cdots+C_{n-2}C_4+C_{n-1}C_3) \end{gather}

卡特兰数的另类递推式有：

\begin{align} &C_n=C_{n-1}\times\frac{4n-2}{n+1}\\ &C_{n+1}=C_n\times\frac{4n+2}{n+2} \end{align}

其中，C1=1C_1=1C​1​​=1 。

为防止符号冲突，记 δ(x,y)=(xy)\delta(x,y)=\begin{pmatrix} x\\y \end{pmatrix}δ(x,y)=(​x​y​​)

Cn=δ(2n,n)n+1=δ(2n,n)δ(2n,n−1)(n∈N)C_n=\frac{ \delta(2n,n) }{n+1}=\frac{\delta(2n,n)}{\delta(2n,n-1)}(n\in N) C​n​​=​n+1​​δ(2n,n)​​=​δ(2n,n−1)​​δ(2n,n)​​(n∈N)

    if a < p and b < p:
        return C(a, b, p)
    return C(a % p, b % p, p) * lucas(a // p, b // p, p) % p

        res = lucas(a, b, p)
