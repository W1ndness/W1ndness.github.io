一、欧拉函数

1.1 欧拉函数公式

欧拉函数 φ(n)\varphi(n)φ(n)​​ 为 [1,n][1,n][1,n]​​ 中与 nnn​​ 互质的数的个数。由离散数学知识，对于一个正整数 NNN​​ ，由算数基本定理，若其标准分解式为 N=p1m1p2m2⋯pkmk(p1<p2<⋯<pk)N=p_1^{m_1}p_2^{m_2}\cdots p_k^{m_k}(p_1<p_2<\cdots<p_k)N=p1m1​​p2m2​​⋯pkmk​​(p1​<p2​<⋯<pk​)​​​ ，则有 φ(n)\varphi(n)φ(n)​ 的值为

\begin{align} \varphi(n)&=N(1-\frac{1}{p_1})(1-\frac{1}{p_2})\cdots(1-\frac{1}{p_k})\\ &=N\prod_{i=1}^k(1-\frac{1}{p_i})\\ \end{align}

那么用这个公式去求欧拉函数的值的时间复杂度为 O(n⋅ai)O(\sqrt n\cdot a_i)O(n​⋅ai​)​

int getEuler(int a) {
    int res = a;
    for (int i = 2; i <= a / i; i++) {
        if (a % i == 0) {
            res = res / i * (i - 1);
            while (a % i == 0) a /= i;
        }
    }
    if (a > 1) res = res / a * (a - 1);
    return res;
}

/** Defination for linear sieve to get Euler
*	int primes[N], cnt;
*	int phi[N];
*	bool st[N];
*/
void getEulers(int n) {
    phi[1] = 1;
    for (int i = 2; i <= n; i++) {
        if (!st[i]) {
            primes[cnt++] = i;
            phi[i] = i - 1;
        }
        for (int j = 0; primes[j] <= n / i; j++) {
            st[primes[j] * i] = true;
            if (i % primes[j] == 0) {
                phi[primes[j] * i] = primes[j] * phi[i];
                break;
            }
            else phi[primes[j] * i] = (primes[j] - 1) * phi[i];
        }
    }
}

快速幂的核心思路是反复平方法，核心思想是：有数 a,k,pa,k,pa,k,p ，预处理出 a20mod  p,a21mod  p,a22mod  p，⋯ ,a2log⁡kmod  pa^{2^0} \mod p,a^{2^1}\mod p,a^{2^2}\mod p，\cdots,a^{2^{\log k}}\mod pa20modp,a21modp,a22modp，⋯,a2logkmodp 的值，则最后我们的目的是将 aka^kak 拆成前面若干项乘积的形式，即假设 ak=a2r1a2r2⋯a2rta^k=a^{2^{r_1}} a^{2^{r_2}}\cdots a^{2^{r_t}}ak=a2r1​a2r2​⋯a2rt​ ，化简指数幂为 ak=a2r1+2r2+⋯+2rta^k=a^{2^{r_1}+2^{r_2}+\cdots+2^{r_t}}ak=a2r1​+2r2​+⋯+2rt​ ，于是显然有 k=2r1+2r2+⋯+2rtk=2^{r_1}+2^{r_2}+\cdots+2^{r_t}k=2r1​+2r2​+⋯+2rt​​ 成立，那么因为底数均为 222 ，则有 k=(r1r2⋯rt)2k=(r_1r_2\cdots r_t)_2k=(r1​r2​⋯rt​)2​ 成立，即指数各项与 kkk​​ 的二进制表示的各个数位一一对应。

那么我们要如何预处理上面的一系列数呢？显然，我们有下面的递推关系成立：

\begin{align} &a^{2^0}=a^1\\ &a^{2^1}=a^{2^0\cdot2}=(a^{2^0})^2\\ &a^{2^2}=(a^{2^1})^2\\ &\cdots\\ &a^{2^{\log k}}=(a^{2^{\log k}-1})^2\\ \end{align}

那么有递推公式 a2i=(a2i−1)2a^{2^i}=(a^{2^{i-1}})^2a2i=(a2i−1)2​​ 成立，我们就能够得出这一系列的数值了。

于是如果 kkk 的二进制表示是 (xmxm−1⋯x2x1x0)2(x_mx_{m-1}\cdots x_2x_1x_0)_2(xm​xm−1​⋯x2​x1​x0​)2​ ，那么我们就可以倒序使用位运算不断拆解 kkk ，由于次数从个位向上是递增的，且二进制表示下非 000 即 111 ，那么也就是 kkk 的二进制表示有多少位，相对应的 a2ia^{2^i}a2i 就要进行平方多少次，能够得到某一位的因子，如果二进制表示下该位为 111 ，则将这个因子与前面的结果相乘，最终便得到了 aka^kak。

typedef long long LL;		// 乘因子可能会爆int所以要开long long

int quickPow(int a, int k, int p) {		// return (a^k % p)
    int res = 1;
    while (k) {
        if (k & 1) res = (LL)res * a % p;
        k >>= 1;
        a = (LL)a * a % p;
    }
    return res;
}

三、扩展欧几里得算法

3.1 裴蜀定理

裴蜀定理的内容：

对任何整数 aaa 、bbb 和它们的最大公约数 ddd，关于未知数 xxx 和 yyy​​ 的线性丢番图方程（称为裴蜀等式）：

ax+by=max + by = max+by=m 有解当且仅当 mmm 是 ddd 的倍数。

求解上述方程中 x,yx,yx,y 的过程即为扩展欧几里得算法。

3.2 扩展欧几里得算法

欧几里得算法

int gcd(int a, int b) {
    return b ? gcd(b, a % b) : a;
}

扩展欧几里得算法

扩展欧几里得算法可以求得裴蜀定理中方程 ax+by=max+by=max+by=m 的 x,yx,yx,y​ 值，算法基于原欧几里得算法进行扩展，算法思想推导如下：

\begin{align} (a,b)=(b,a\text{ mod }b)=d\\ &\Leftrightarrow \left\{ \begin{array}{c} ax+by=d\\ bx+(a\text{ mod }b)y=d\\ \end{array}\tag{*}\label{c} \right.\\ &\text{where }a\text{ mod b}=a-\lfloor\frac{a}{b}\rfloor \cdot b\\ &\text{Now Eq\ref{c}(b)}\\ &\Rightarrow bx+(a-\lfloor\frac{a}{b}\rfloor \cdot b)y=d\\ &\text{Swap }x\text{ and }y\text{ to simplify calculate and corresponding to eq(*)(1)}\\ &\Rightarrow by+(a-\lfloor\frac{a}{b}\rfloor \cdot b)x=d\\ &\Leftrightarrow ax+b(y-\lfloor\frac{a}{b}\rfloor \cdot b)=d\\ &\Leftrightarrow \begin{cases} x'=x\\ y'=y-\displaystyle\lfloor\frac{a}{b}\rfloor \cdot b \end{cases}\\ \end{align}

则得到通过递归写出的代码如下：

int exgcd(int a, int b, int &x, int &y) {
	if (!b) {
        x = 1, y = 0;
        return a;
    }
    int d = exgcd(b, a % b, y, x);
    y -= a / b * x;
    return d;                                                                                                                                                        
}

def getEulers(n):
    global primes, cnt
    for i in range(1, n + 1):
        if i == 1:
            phi[1] = 1
            continue
        if not st[i]:
            primes.append(i)
            phi[i] = i - 1
        for prime in primes:
            if prime * i > n:
                break
            st[prime * i] = True
            if i % prime == 0:
                phi[prime * i] = phi[i] * prime
                break
            else:
                phi[prime * i] = phi[i] * (prime - 1)

    phi = [0] * N
    getEulers(n)
    res = 0
    for i in range(1, n + 1):
        res += phi[i]
    print(res)

    int n;

    cin >> a1 >> m1;

    for (int i = 0; i < n - 1; i++) {
        LL a2, m2;
        cin >> a2 >> m2;

        LL d = exgcd(a1, a2, k1, k2);
        if ((m2 - m1) % d) {
            hasAnswer = false;
            break;
        }
        k1 *= (m2 - m1) / d;
        LL t = a2 / d;
        k1 = (k1 % t + t) % t;
        m1 = a1 * k1 + m1;
        a1 = abs(a1 / d * a2);
    }
    if (hasAnswer) cout << (m1 % a1 + a1) % a1;
    else puts("-1");
